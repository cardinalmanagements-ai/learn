۰٪

📚 مجموعه چیه؟

🎯 بیا قبل از هر چیز، با دو تا مثالِ ساده از زندگیِ خودت شروع کنیم!
مثال ۱
⚽

تیمِ فوتبالِ مدرسه

آقای مربی یه برگه دستش می‌گیره و اسمِ بازیکن‌های اصلی رو می‌نویسه:

تیم = ‌{ علی، حسن، محمد، رضا، مهدی }

✅ اسمِ هر بازیکن کاملاً مشخصه

✅ هیچ اسمی دو بار نوشته نشده

✅ اگه ترتیبشون رو عوض کنیم، باز همون تیمه

مثال ۲
🎵

پلی‌لیستِ موسیقی

توی گوشیت یه پلی‌لیست ساختی به اسم «آهنگ‌های موردِ علاقه‌ام»:

پلی‌لیست = ‌{ آهنگِ ۱، آهنگِ ۲، آهنگِ ۳ }

✅ مشخصه چه آهنگ‌هایی توشه

✅ یه آهنگ دو بار اضافه نمی‌شه

✅ ترتیبِ پخش، خودِ لیست رو عوض نمی‌کنه

🤔 خب، این دو تا چی مشترک دارن؟

هر دو یه دسته‌ی مشخص و بدونِ تکرار از چیزایی‌ان. اعضاش کاملاً معلومه و ترتیب‌شون هم مهم نیست.

✨ تو ریاضی به این می‌گن: «مجموعه»

📌 حالا تعریفِ رسمی

تو زندگیِ روزمره از واژه‌هایی مثل «دسته»، «گروه»، «مجموعه» زیاد استفاده می‌کنیم. مثلاً «گروهی از بازیکنان وارد ورزشگاه شدند» — این جمله مشخص نمی‌کنه چه بازیکنانی! ولی تو ریاضی، «مجموعه» باید کاملاً دقیق و بدون ابهام باشه.

📖 تعریف: مجموعه یعنی دسته‌ای از اشیای مشخص (بدون ابهام) و متمایز (بدون تکرار).

🎮 یه استعاره‌ی جذابِ دیگه: اسکوادِ بازیِ آنلاین

تصور کن داری یه بازیِ آنلاینی مثل کالاف‌دیوتی یا فورتنایت می‌کنی و باید یه اسکواد (تیم) از دوستات بسازی:

👾

مجموعه = اسکوادِ تو

یه تیم ساختی به اسمِ A. لیستِ گیمرایی که توی این اسکواد عضوَن، می‌شه مجموعه‌ی A.

🕹

عضو = هر گیمر در تیم

هر بازیکنی که آیدیش (یوزرنیمش) تو لیستِ اسکواد ثبت شده، یه «عضو» مجموعه‌ست.

⚔️

شرطِ ورود = توصیفِ مجموعه

کاپیتانِ تیم یه قانون داره: مثلاً «فقط کسایی که لِوِل‌شون بالای ۵۰ هست می‌تونن عضو بشن».

🛡

مرزِ تیم = نمودار وِن

یه دایره دور آیدی گیمرای تیم می‌کشیم. هرکی بیرونِ این دایره‌ست، تو اسکوادِ ما نیست.

📖 مثال‌های بیشتر از زندگیِ واقعی

✅ مجموعه‌ست: دانش‌آموزانِ کلاسِ نهمِ شما

✅ مجموعه‌ست: روزهای هفته → {شنبه, یکشنبه, ..., جمعه}

✅ مجموعه‌ست: اعداد طبیعیِ یک‌رقمی → {1,2,3,4,5,6,7,8,9}

❌ مجموعه نیست: «چند تا از بازیکنان تیم ملی» — کدوماشون؟! نامشخصه.

❌ مجموعه نیست: «دانش‌آموزان خوش‌اخلاق» — خوش‌اخلاقی نسبیه!

🔑 نکته‌ی کلیدی: مجموعه‌ها رو با حروف بزرگ انگلیسی (A, B, C, ...) نام‌گذاری می‌کنیم و اعضاش رو داخل { } می‌نویسیم.
مثال: A = {2, 3, 5}

📏 سه قانونِ طلاییِ مجموعه

هر مجموعه‌ی ریاضی باید این ۳ شرط رو داشته باشه. روی هر کارت کلیک کن تا مثال‌هاش رو ببینی:

1 قانون مشخص بودن

توصیفِ اعضای مجموعه باید کاملاً واضح باشه. هیچ ابهامی نباید باشه. دو نفر مختلف نباید بتونن دو جوابِ متفاوت ازش دربیارن.

📂 نمونه‌ها رو ببین
✅ «اعدادِ زوجِ بین ۱ تا ۱۰» → کاملاً مشخصه: {2, 4, 6, 8}
❌ «چهار شاعرِ ایرانی» → کدوم چهار تا؟! ایران هزاران شاعر داره!
❌ «۳ شمارنده‌ی زوجِ ۶۰» → احمد می‌تونه {4,6,10} بنویسه و رضا {6,10,12} — هر دو درستن، ولی متفاوت!

2 قانون متمایز بودن

هیچ عضوی نباید تکرار بشه. اگه یه عنصر دو بار نوشته شده باشه، یکیش رو حذف می‌کنیم.

📂 نمونه‌ها رو ببین
{3, 3, 4} = {3, 4}
{1, 2, 1, 4, 5} = {1, 2, 4, 5}
{5, 6, 5, 7} = {5, 6, 7}

3 قانون بی‌اهمیّتیِ ترتیب

ترتیبِ نوشتنِ اعضا هیچ اهمیتی نداره. هر چیدمانی همون مجموعه رو می‌سازه.

📂 نمونه‌ها رو ببین
{2, 3, 5} = {5, 3, 2} = {3, 5, 2}

تو اسکوادِ بازی، مهم نیست کی اوّل به تیم اضافه شد. مهم اینه که الان کیا تو تیمن!

💡 نتیجه‌ی نهایی: برای اینکه چیزی «مجموعه» باشه، باید بشه با قطعیّت گفت هر چیزی عضوِ این مجموعه هست یا نه.

🔢 شمارنده‌های ۶۰ — کارگاهِ تعاملی

شمارنده‌های عددِ ۶۰ این‌ها هستن. روی دکمه‌های زیر کلیک کن تا فیلتر کنی و مجموعه‌های A، B، C، D رو بسازی:

شمارنده‌های ۶۰ = {1, 2, 3, 4, 5, 6, 10, 12, 15, 20, 30, 60}

📊 چهار مجموعه‌ی مهم ۶۰

مجموعه A (شمارنده‌های اول): A = {2, 3, 5} — تعداد اعضا: ۳

مجموعه B (شمارنده‌های مرکّب): B = {4, 6, 10, 12, 15, 20, 30, 60} — تعداد اعضا: ۸

مجموعه C (نه اول، نه مرکّب): C = {1} — تعداد اعضا: ۱

مجموعه D (دو رقمی): D = {10, 12, 15, 20, 30, 60} — تعداد اعضا: ۶

⚠️ تله‌ی مهم: عددِ ۱ نه عددِ اوّله، نه عددِ مرکّب! یه استثناست. به همین خاطر مجموعه‌ی C فقط یه عضو داره.

∈ ∉ نمادهای عضویّت

دو تا نماد خیلی مهم تو مجموعه‌ها داریم:

📍 برای گفتنِ «x عضوِ مجموعه‌ی A است» می‌نویسیم: x ∈ A

📍 برای گفتنِ «x عضوِ مجموعه‌ی A نیست» می‌نویسیم: x ∉ A

🧠 ترفندِ یادآوری: نمادِ ∈ شبیهِ حرفِ E انگلیسیه = Element (عضو). خط روش = نه!

✏️ تمرین: کدوم درسته؟

فرض کن مجموعه: A = {a, b, 5, 7}

🏗️ مجموعه است یا نیست؟

به هر توصیف نگاه کن و تشخیص بده آیا یه مجموعه‌ی معیّن و یکتا می‌سازه یا نه:

⭕ نمودار وِن — ساخت تعاملی

یه عضو وارد کن، روی دکمه «اضافه کن» بزن. ببین چطور وارد دایره می‌شه:

A = {}
📌 نکته: دایره فقط یه شکل از نمایشه. می‌تونی هر منحنیِ بسته‌ای بکشی — مثلاً بیضی، مثلث، یا حتی یه شکلِ نامنظم. مهم اینه که بسته باشه.
